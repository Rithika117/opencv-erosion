## Morphological Erosion using OpenCV

This project demonstrates the implementation of Erosion using Python and OpenCV. Erosion is a morphological operation used to remove small white noises and shrink object boundaries in an image.

## Aim

To implement erosion operation using OpenCV.

## Software Required
Python 3.7
OpenCV
NumPy
Jupyter Notebook / VS Code
## Operation Performed
Create text/image

Create structuring element (kernel)

Apply erosion using cv2.erode()

Display the eroded image

## Output
Original Image

The original image contains the input text/object before applying morphological operations.

Eroded Image

After applying erosion using cv2.erode(), the white regions of the image become thinner. Small noises are removed, and the boundaries of the objects shrink slightly based on the kernel size and iterations used.

## Result

Thus, the erosion operation was successfully implemented using Python and OpenCV.
